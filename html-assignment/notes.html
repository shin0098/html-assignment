Headings

We can use the <h1></h1> tag for the most important heading of the page. For the next level of heading we have <h2></h2>, <h3></h3>, <h4></h5 >, and <h6></h6>. The browser will style each heading differently, stating with the biggest size for h1 then h2 and so on.

Div

The div tag defines a division or a section in an HTML document. The tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.

Button

The button HTML element is an interactive element activated by a user with a mouse, keyboard, finger, voice command, or other assistive technology. Once activated, it then performs a programmable action, such as submitting a form or opening a dialog.
